# Blog Post Starter

Copy this template into a new file under src/content/blog, for example:
- src/content/blog/my-new-post.md

```md
---
title: "Your Post Title"
meta_title: "Your Post Title | Kanso Blog"
description: "One sentence summary of the post for search and social previews."
image: "/images/your-cover-image.png"
date: 2026-05-06T09:00:00Z
categories: ["career"]
tags: ["topic-one", "topic-two"]
draft: false
---

Intro paragraph.

## Main Point

Your content here.

![Optional inline image](/images/your-inline-image.png)
```

## SEO Checklist

- Keep title specific and human-readable.
- Keep description around 140-160 characters.
- Use a real cover image in /public/images.
- Include 1-3 categories and a few relevant tags.
- Set draft: false when ready to publish.
